# دعوة ديت ديتون

مشروع دعوة تفاعلية إلى الريوق، مبنية كـ Static Site داخل GitHub Pages.

## الرابط

بعد النشر:

https://7htma.github.io/wedding-invitation-hatem-maather/date-daiton/

## المشاهد

1. افتتاحية الستارة: شخصية حاتم تفتح الدعوة.
2. رسالة الدعوة: دعوة زوجية إلى الريوق.
3. رحلة البيت إلى Bound café: سيارة ومسار تفاعلي.
4. كلاس قوانين ديت ديتون: سبورة طباشير وثلاثة قوانين.
5. صفحة الموافقة والاحتفال.

## التعديلات السريعة

- النصوص داخل `index.html`.
- الألوان والحركات داخل `css/style.css`.
- القوانين والحركات داخل `js/app.js`.
