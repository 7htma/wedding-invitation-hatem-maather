## الملفات الرئيسية

- `index.html` صفحة دعوة الزفاف الرئيسية
- `admin.html` صفحة إدارة تأكيد حضور دعوة الزفاف
- `date-daiton/index.html` صفحة دعوة ديت ديتون
- `date-daiton/css/style.css` ستايل ديت ديتون
- `date-daiton/js/app.js` تفاعلات ديت ديتون
- `.github/workflows/pages.yml` نشر تلقائي على GitHub Pages
